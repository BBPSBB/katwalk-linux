Motion capture

Records the raw foot & body-sensor data for a set of labeled movements, so the locomotion mapping can be built from what your steps actually look like. Hit start, then go strap in and put the shoes on - a big lead-in timer counts you down before the first action.

Make sure the driver shows streaming first (dashboard). For each step action: step out → pause → let it slide back → pause → repeat a few times.
